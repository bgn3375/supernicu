Bono Design System

Sistemul vizual care unește Landing, Dublin (dashboard) și Ghiduri. Bej cald (R>G>B), pink #EE4379 ca brand, Inter 600 pentru titluri și Fraunces italic pentru accente. Grid 28px, line @ 6%.

Culori

01 · Locked

Patru suprafețe. Două bej-uri (calde, cu o tentă imperceptibilă spre roz), plus alb și un singur dark.

Un singur ink închis peste toate suprafețele light. Variația tipografică vine din ierarhie (size + weight), nu din schimbarea darkness-ului per surface.

11 reguli de utilizare
  1. Tracking negativ obligatoriu pe orice text peste 24px.
  2. Source Serif 4 exclusiv pe T20 (body blog) — niciodată în UI.
  3. Fraunces exclusiv pe T-em și T-actor — niciodată altundeva.
  4. tabular-nums obligatoriu pe toate cifrele financiare și T-num-inline.
  5. T-em — maxim 3 cuvinte per heading, exclusiv T60/T48/T40, culoare pink, weight 600.
  6. T-em letter-spacing — identic cu heading-ul de bază, nu se reduce suplimentar.
  7. T12 eyebrow — exclusiv label static, nu pe touch targets interactive.
  8. T-actor weight 500 — nu 400, italic subțire la 13px dispare pe non-Retina.
  9. Tn40 și Tn28 ambele weight 700 — diferența de size e suficientă pentru ierarhie.
  10. Nu folosim zecimale în dimensiuni de font (13.5px, 14.5px etc.).
  11. T60.tight — exclusiv pentru titluri single-line garantate, nu ca default.

Grid pattern

03b · Texture

Dot grid subtil, ink @ 15% alpha, 32px spacing. Aplicat pe fundalul bej-0 al tuturor paginilor (Landing, Dublin, Blog, System) prin clasa .has-grid pe <body>.

Tokens
--grid-color   rgba(17, 13, 16, 0.15)   /* ink @ 15% */
--grid-size    32px                       /* spacing punct la punct */
--grid-radius  1px                        /* radius dot (Ø 2px) */
--grid-edge    1.5px                      /* tranziție sharp */
Live preview
Decizii & rationale
  • Ink, nu auriu — la luminanță scăzută, ink dă contrast perceptual stabil cross-device la alpha mic.
  • Dots, nu lines — pattern „graphite paper / Notion canvas", nu „spreadsheet / engineering blueprint".
  • 0.15 alpha — pragul „vizibil clar fără să strige". Sub 0.10 dispare; peste 0.20 devine pattern.
  • 32px spacing — densitate mid-range. 24px = prea „graph paper", 48px = prea aerisit.
  • No mask — textura e deja subtilă; mask-ul ar lăsa goluri.

Inputs

05 · Forms
Reguli
Padding interior (orizontal)

Egal cu radius-ul: 16px pe input default (h 40, r 20), 20px pe input .lg (h 44, r 20). Textul începe unde se termină curba — aliniat optic.

Padding exterior (vertical)

Gap label → input → help: 4px. Densitate strânsă, label pare ancorat de câmp. Între câmpuri în formular: 16px (margin-bottom pe .field).

Aliniere label → text

Label și help padding-left 16px (= padding interior input md). Label, text input și help încep de la aceeași coloană verticală.

Tipografie

Label: Inter 12 / 500 / sentence case / tracking 0.04em, color fog (label-quiet — discret, nu strigă). Text input: Inter 14 / 400 (md) sau 16 / 400 (lg), color ink. Help: Inter 12 / 400, color fog.
